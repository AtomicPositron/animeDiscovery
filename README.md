## Installation

Make sure **NPM** and **GIT** is installed on your computer. From your command line

**Clone the Repository**

```bash

$ git clone https://github.com/MichaelIbia12/animeDiscovery.git

```

## Usage

```bash

$ npm run dev

or 

$ pnpm dev

```
